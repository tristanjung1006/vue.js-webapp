# vue-webapp
- Vue CLI를 이용한 프로젝트 구성
- 컴포넌트 기반 설계
- 컴포넌트 구조화 및 통신 구현
- ES6, Vuex

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
